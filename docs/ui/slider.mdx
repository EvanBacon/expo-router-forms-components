A slider component that allows users to select a value or range from a continuous or discrete set of values. Uses gesture-based interaction on native platforms and Radix UI on web.

## Installation

<InstallBlock packages="@radix-ui/react-slider" />

## Usage

```tsx
import { Slider } from "@/components/ui/slider";
```

## Platform Differences

### Native (iOS/Android)

- Uses `react-native-gesture-handler` for touch interactions
- Animated with `react-native-reanimated` for smooth thumb movement
- Thumbs scale up slightly when pressed for visual feedback
- Supports `minStepsBetweenThumbs` to prevent thumbs from overlapping

### Web

- Uses Radix UI's Slider primitives
- Full keyboard navigation support (arrow keys to adjust value)
- Hover and active states with smooth transitions
- Focus-visible ring for accessibility
- Supports all Radix slider props

## Accessibility

- Uses proper `slider` role for screen readers
- Keyboard accessible: use arrow keys to adjust value
- Focus states are clearly visible
- Value changes are announced to assistive technologies
- Supports `aria-label`, `aria-labelledby`, and `aria-describedby` props
